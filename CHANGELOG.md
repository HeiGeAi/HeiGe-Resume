# Changelog

本项目所有重要变更记录于此。

## [1.0.0] - 2026-06-01

首个版本。

### 核心
- 原创方法论「6 秒决定去留」，五道锻造工序：一句话定位 / 写结果不写职责 / 6 秒扫描 / 一页纪律 / 反模板体检。
- 四步使用流程：定调 → 搭骨架（定位句 + 选战功）→ 写生产级代码 → 反模板体检。

### 设计资产
- `references/aesthetic-directions.md`：5 种气质方向库（克制专业 / 工业硬核 / 高定优雅 / 设计感张扬 / 温暖简洁），每种给出色彩 / 字体 / 排版打法。
- `references/resume-production-spec.md`：单文件 HTML 简历硬规格。PDF 优先（A4 / Cmd+P 完美导出）+ ATS 友好（文字可选中 / 不嵌图 / 不用进度条 / 阅读顺序正确）+ 中文字体兜底，生产铁律从第一天内置。
- `references/content-templates.md`：内容模板。一句话定位公式、结果不写职责改写公式、各职业各阶段战功写法。
- `references/anti-template-checklist.md`：反模板体检清单，出货前逐条过。

### 样例画廊（同一套方法论 × 五种职业气质，全部虚构人物保护隐私）
- `examples/heige-resume.html`：资深后端工程师。工业 / 硬核。1 页。
- `examples/pm-resume.html`：产品经理。克制 / 专业。1 页。
- `examples/designer-resume.html`：设计师。设计感 / 张扬。1 页。
- `examples/newgrad-resume.html`：应届毕业生。温暖 / 简洁。1 页。
- `examples/executive-resume.html`：技术高管。高定 / 极简。2 页。

全部为零依赖单文件，A4 排版，文字可选中（ATS 友好），可 Cmd+P 完美导出 PDF，中文带系统字体兜底。每份均经真实 PDF 导出验证页数与文字可提取性。
