# 气质方向库（Resume Aesthetic Directions）

定调时按**目标岗位**选方向。多数岗位偏克制专业，设计/创意岗可以张扬。
原则：简历的气质要服务于「让最强信号被看到」，不是炫技。

每种方向给出可直接落地的打法：色彩、字体、排版、强调怎么做。

---

## 1. 克制 / 专业（Swiss · 默认首选）

气质：纪律、清晰、可信、信息优先。适合大多数岗位，尤其技术、金融、咨询、产品、大厂。

- **色彩**：黑白灰为主 + 一个克制的强调色（深蓝 / 墨绿 / 暗红），只用在姓名下划线、小标题、关键数字。
- **字体**：Inter / Helvetica / 思源黑体，严格字号阶梯。英文名可用一点衬线提气质。
- **排版**：单栏或主次分明的偏栏（左窄右宽），左边界对齐，时间右对齐到一条线。大量克制留白。
- **强调**：只给公司名、关键数字、最强一条加粗或上强调色。
- **照片**：多数情况不放。

## 2. 工业 / 硬核（Brutalist）

气质：硬、有态度、自信。适合资深工程师、技术专家、有锋芒的角色、敢秀的人。

- **色彩**：近黑 + 一个电光强调色（电光橙 / 酸绿），高对比。或纯黑白 + 粗黑线。
- **字体**：超粗无衬线（Archivo / Helvetica Bold / 思源黑体 Heavy）做姓名和标题 + 等宽体做标注（时间、技能、链接）。
- **排版**：粗黑分割线、裸露网格、左对齐、大字号姓名占顶部一条。等宽体标注像终端。
- **强调**：关键数字用强调色或反白块。
- 注意：再硬也要守 ATS（文字可选中）和一页纪律。

## 3. 高定 / 优雅（Editorial）

气质：精致、有分量、资深感。适合高管、资深管理者、咨询、品牌、需要分量感的角色。

- **色彩**：纸感米白 / 暖灰 + 一个深色（墨蓝 / 炭黑 / 酒红）+ 至多一点金。低饱和。
- **字体**：高质量衬线（Playfair / 思源宋体）做姓名和小标题 + 干净无衬线正文。
- **排版**：杂志感，大量留白，姓名大字 + 一条细分割线，小标题用小号大写字距拉开。
- **强调**：靠字体和留白分层级，不靠颜色堆。一句定位用大号衬线。

## 4. 设计感 / 张扬（Expressive）

气质：有创意、敢表达、本身就是作品。适合设计师、创意、前端、品牌、需要展示审美的角色。

- **色彩**：可以用一个大胆主色块（但别滥）、撞色点缀；左栏色块 + 右栏白底是常见手法。
- **字体**：有性格的字体组合，可混搭，但正文仍要清楚可读。
- **排版**：可以有色块分区、几何元素、清晰的视觉节奏。本身就在展示排版能力。
- **强调**：用色块、留白、对比讲层级。
- 注意：再有设计感，关键信息也必须是真文字（ATS），别把名字做成图形。

## 5. 温暖 / 简洁（Warm · Minimal）

气质：干净、亲切、不端着。适合应届生、转行者、内容/运营/教育/HR、经历不多需要显得清爽的人。

- **色彩**：白底 + 一个柔和强调色（暖橙 / 柔绿 / 雾蓝），低饱和。
- **字体**：圆润无衬线或温暖的字体，舒服好读。
- **排版**：清爽单栏，大留白，少即是多。经历不多时，靠留白和清晰层级显得专业，而不是硬塞。
- **强调**：克制，把有限的几条做扎实，突出项目和潜力。

---

## 跨方向通用的"反平庸"硬规则

无论选哪个方向，这几条都适用：

1. **一句话定位**放在最显眼处，姓名之下第一眼能看到。
2. **强制字号阶梯**：姓名 > 定位 > 小标题 > 公司职位 > 正文 > 标注，层级清楚。
3. **强调克制**：加粗 / 强调色只给最该被看到的，别全篇开花。
4. **守 ATS**：关键信息一律真文字可选中，不嵌图、不用进度条。
5. **一页纪律**：先筛后排，宁可少而精，不要多而满。
