# 反模板体检清单（Resume Anti-Template Checklist）

出货前**强制**逐条过。命中任何一条，回去改。
这不是建议，是交付门槛。

---

## 一、致命烂简历特征（命中=必改）

一眼就能看出「AI 套模板 / 流水账」，出现一个就回去改：

- [ ] **烂大街双栏模板**：那个人人都在用的「现代简约」左深右白双栏，跟八百份一样。
- [ ] **通篇职责无结果**：满屏「负责 / 参与 / 协助」，没有一个结果、一个数字。
- [ ] **没有定位句 / 定位是废话**：顶部没有一句话定位，或写成「求职意向：XX」「追求卓越」这种口号。
- [ ] **技能进度条 / 五角星**：用进度条、五角星、百分比表达「精通 Python」，信息量为零。
- [ ] **关键信息嵌图**：姓名、电话、邮箱做成图片 / 图标字形，机器读不出。
- [ ] **花哨盖过内容**：配色、图标、装饰抢了戏，内容反而看不清。
- [ ] **注水填满**：为了填满一页 / 两页，把不相关的经历、课程、爱好都堆上。

## 二、一句话定位体检

- [ ] **姓名下方第一眼有定位句吗？**
- [ ] **这句话是事实还是口号？**（要「五年后端，扛过日活千万支付系统」，不要「追求技术卓越」）
- [ ] **塞进了你最能打的一个事实 + 数字 / 体量吗？**
- [ ] **针对目标岗位调过强调点吗？**

## 三、战功体检（结果不写职责）

- [ ] **每条经历是结果还是职责？**（做成了什么 vs 负责什么）
- [ ] **有数字吗？**（百分比 / 金额 / 倍数 / 人数 / 时长）量化不了的有可验证结果吗？
- [ ] **动词是有担当的吗？**（重构 / 主导 / 从 0 搭 vs 负责 / 参与 / 协助）
- [ ] **最强的战功在最前面吗？**

## 四、6 秒扫描体检

- [ ] **扫一眼 6 秒，能抓到三个重点吗？**（你是谁 · 最强一条 · 能不能用）
- [ ] **最强的经历 / 公司 / 数字，在视线第一落点（顶部、左上）吗？**
- [ ] **层级清楚吗？** 还是所有条目一样重、什么都加粗？

## 五、一页纪律体检

- [ ] **控制在一页了吗？**（资深最多两页，且第一页能独立成立）
- [ ] **对齐有纪律吗？** 左边界统一、时间右对齐到一条线？
- [ ] **字号阶梯统一吗？** 姓名 / 定位 / 标题 / 正文层级清楚？
- [ ] **是先筛后排，还是硬塞 / 注水？**

## 六、ATS 与导出体检（投递前必做）

- [ ] **所有文字可选中、可复制吗？**（在浏览器里试着选中姓名、邮箱、经历）
- [ ] **关键信息没有嵌图、没有靠图标承载吧？**
- [ ] **阅读顺序逻辑正确吗？**（DOM 顺序 = 人读的顺序，双栏没读串）
- [ ] **小标题是机器认得的标准词吗？**（工作经历 / 项目经历 / 教育 / 技能）
- [ ] **Cmd+P 导出的 PDF 是干净的一页吗？** 没有多出空白页、没有把一条经历从中间切断？
- [ ] **中文字体有系统兜底吗？** 换电脑 / 离线也正常？

## 七、机器这关与诚信体检（投递前必做）

2026 年简历先过机器：被关键词搜到、被 AI 按岗位打分排序，过了才到人眼。这关也要过。

- [ ] **分清是哪份了吗？** 扔进简历库等人搜的泛用版，关键词铺到位了吗；对着具体岗位投的特化版，对岗改过了吗？
- [ ] **用上岗位的原词了吗？** 用招聘要求里的词复述经历（岗位匹配在 AI 打分里约占四成）。
- [ ] **资历和体量写出来了吗？** 担当动词 + 团队规模 + 业务体量 + 决策层级？
- [ ] **近期经历靠前吗？** 别让旧成果盖过近两三年的新成果。
- [ ] **没作弊吧？** 没有白字塞关键词、没有夹带指令糊弄 AI。这些会被检测并惩罚，本系统拒绝执行。
- [ ] **每条经历经得起追问吗？** 面试官问「这个数字怎么来的、你具体做了啥」，你答得上算法、来源、你的角色？

两条都"会"，才能投。
